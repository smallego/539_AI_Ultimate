# Roadmap

## V7.2

- Harden CI with coverage reporting and artifact upload.
- Add Docker healthcheck and production logging defaults.
- Add user-facing export for prediction history and explain reports.
- Add dashboard empty-state polish for first-run environments.
- Add API response schema documentation.

## V7.3

- Add role-based settings for local deployment environments.
- Add scheduled background refresh jobs for update, backtest, and dashboard rebuild.
- Add read-only admin page for logs and recent API failures.
- Add release packaging for Windows desktop users.

## V8

- Separate the Web API from long-running worker jobs.
- Add queue-based task execution for Run All, Prediction, Backtest, and Dashboard generation.
- Add optional multi-database deployment profile.
- Add model observability dashboards for drift, stability, and long-term performance.
- Prepare hosted deployment patterns while keeping local-first operation.
