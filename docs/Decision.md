# Decision

Decision Center summarizes model signals into a dashboard-level recommendation.

## Inputs

- AI score
- Final score
- Learning signal
- Recent ROI and hit rates
- Hot/cold overlap
- Trend signal

## Output

- Decision Score
- Confidence
- Risk
- Recommendation
- Decision reasons

## API

```text
GET /api/decision
```

## Release Candidate Rule

V7.1.3 does not change the Decision algorithm.
